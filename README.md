# Finance by Kubi 💰

A responsive and interactive personal finance tracking web application. Built to help users easily monitor their income, expenses, and overall account balance in one clean dashboard.

## 🚀 About The Project

"Finance by Kubi" is a robust Single Page Application (SPA) designed for straightforward financial management[cite: 46]. Whether you need to log a quick coffee purchase or track your monthly salary, this tool provides a clear overview of your financial health. 

### ✨ Key Features

**Financial Dashboard:** Instantly view your total income, total expenses, and current balance in a visually distinct layout[cite: 48, 49, 50].
**Transaction Management:** Seamlessly add, edit, or delete transactions (both incomes and expenses)[cite: 30, 39, 72].
**Custom Categories:** Create and assign custom categories to your transactions for better organization[cite: 123, 134].
**Dynamic Filtering & Sorting:** Filter your transaction history by specific categories[cite: 56, 59, 92]. [cite_start]Sort your data by Date, Description, or Amount with a single click[cite: 52, 54, 64, 100].
**Interactive UI:** Real-time DOM updates and smooth Bootstrap-powered modal windows for data entry[cite: 1, 22].
**Instant Feedback:** Built-in toast notifications inform you immediately about successful actions or errors[cite: 74, 112].

## 🛠️ Built With

This project leverages the modern .NET ecosystem and standard web technologies:

**Frontend:** C#, Blazor (Interactive Server), HTML, CSS[cite: 1, 138, 139].
**UI Framework:** Bootstrap 5 & Bootstrap Icons for a responsive and clean user interface[cite: 51, 138].
**Backend:** C#, .NET[cite: 1, 45].
**Database & ORM:** SQLite database managed via Entity Framework Core for reliable and lightweight data storage[cite: 1, 143].
